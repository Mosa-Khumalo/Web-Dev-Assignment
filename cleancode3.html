Laptop Basics

WHAT IS A LAPTOP?
A laptop is a portable, compact personal computer designed for mobile use.

EXTERNAL HARDWARE COMPONENTS:

  • Display Screen: The visual component of a laptop that can be either an LCD or an LED. It allows users to view information and execute tasks.
  • Keyboard: An input device that is used to click the keys and to enter texts, numbers, and symbols. It also acts as the main interface for interacting with the operating system and applications.
  • Touchpad: This is a tactile sensor where you slide your finger to move the cursor onto the screen; it acts as a direct replacement for a mouse.
  • Webcam/Microphone: Used real-time video communication, including video calls and virtual meetings.
  • Power Button:Turns the laptop on and off, acting as a signaling device. It is typically located on the top side of the laptop.
  • Speakers: Integrated speakers are used for listening to music and allow users to hear system alerts.
External Ports

USB Ports: Connect peripherals like flash drives.

HDMI Port: Connect to external displays.

Audio Jack: Connect headphones or microphones.

Keyboard and Functions
1. Keyboard Explanation

A keyboard is an input device used to type letters, numbers, and commands into a computer, making it essential for user interaction.

2. Key Types & Functions

It includes different types of keys with specific functions. Control keys like Ctrl, Alt, and Shift are used for shortcuts, while navigation keys such as Home, End, and Page Up/Down help move through documents. Function keys (F1–F12) perform tasks like opening help menus, refreshing pages, and enabling full-screen mode. Common keyboard layouts include QWERTY (the most widely used), AZERTY, and DVORAK. Keyboard shortcuts, such as Ctrl + C (copy), Ctrl + V (paste), and Ctrl + S (save), help users complete tasks more quickly and efficiently. There are also various types of keyboards, including membrane, mechanical, chiclet, ergonomic, and wireless, each designed for different needs and preferences. Practicing good typing habits—such as using the home row keys, maintaining proper posture, and typing regularly—can improve speed and accuracy. Overall, understanding how keyboards work helps users become more efficient and comfortable when using a computer.

Mouse & Touchpad
1. Background Information

Optical mice with IntelliEye tech detect minor surface differences.

2. Choosing a Mouse
  • Size & shape
  • Wheel mouse reduces movement
  • Extra buttons optional
3. Description of a Mouse

Right Click: Menus & quick actions

Left Click: Selecting & interacting

Scroll Wheel: Scrolls pages & tabs

4. The Touchpad

Flat rectangular surface below keyboard for finger gestures.

5. Touchpad Actions
  • Slide → moves cursor
  • Tap → click
  • Two-finger swipe → scroll
  • Pinch → zoom
💡 Tip

Clean mouse if movement is jerky.
